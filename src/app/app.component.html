Blackbaud Software Engineer Summer Intern 2018, 2019

For the past 2 summers I have worked at Blackbaud, a large provider of nonprofit software solutions. There I worked on their core user experience framework team SKY UX, developing open-source front-end Angular components and internal back-end services using Azure solutions with .NET Core. Several teams throughout the company used these components and services for developing multiple products.

During my time at Blackbaud I worked on 2 major products. The first was a full-stack UI configuration service that preserves the states of several components across user sessions. This project involved creating a library to interface with an Azure App Service which stored user configurations in Azure CosmosDB and interpreting these configurations in the UI. The second project was a complex notification system to accept several types of notifications, determine their delivery type, and handle pushing real-time updates to users that are logged in. This project involved creating 2 microservices and connecting CosmosDB and Azure SignalR to store notifications and push them to users over a WebSocket connection.

At Blackbaud, I gained a better understanding of building back-end services, microservice architecture, and the power of Microsoft's Azure solutions.

Blackbaud SKY UX Good Done Great Software Engineer Summer Intern 2015, 2016, 2017

I spent 3 summers working at Good Done Great, a local startup focused on developing corporate social responsibility solutions for large Fortune 500 firms. As a rising high school sophomore with little experience, I quickly learned web development with AngularJS and began working on an Agile team to create a new product.

Over the course of my experience with Good Done Great, I worked on major projects such as a new graphical dashboard built with amCharts and a new cross-platform mobile application built with Ionic. I also spent time testing and resolving bugs and mentoring new interns.
